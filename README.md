# Book Store Manager

## Overview
This project is a Windows application developed in C# using .NET for managing a bookstore.

It supports inventory management, order handling, and order tracking.

This software is designed for bookstore managers, covering the full workflow from managing inventory and orders to tracking deliveries and system management.

- Import new books to the store  
- Create and manage customer orders  
- Track and archive information about orders and store activity  

## Features

- View books with relevant information (image, summary, price)  
- Simple shopping cart – edit order lines, view total price, cancel or proceed with the order  
- Order form – fill in order details and receive an order ID  
- Starred items – save items for later  
- Order status tracking  
- Simple and clean user interface  

### Manager functionality

- Book management – full control over the book data  
- Order management – update, track, and remove orders if needed  
- Archive for completed orders  

The system includes search and filtering functionality across different windows.
